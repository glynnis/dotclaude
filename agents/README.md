# Agents

Custom subagent configurations that extend Claude Code with specialized roles. Agents are invoked automatically when their description matches the task, or explicitly via the Agent tool.

Agents sourced from [VoltAgent/awesome-claude-code-subagents](https://github.com/VoltAgent/awesome-claude-code-subagents).

---

## Writing & content

**`ai-writing-auditor`** — Audits content for AI writing patterns and rewrites text to remove them.

**`content-quality-editor`** — Use before publishing any AI-generated content. Strips AI writing patterns, then performs a final quality pass.

**`content-marketer`** — Develops content strategies, creates SEO-optimized marketing content, and executes multi-channel content campaigns.

**`technical-writer`** — Creates, improves, or maintains technical documentation including API references, user guides, and SDK documentation.

## Design & UI

**`ui-designer`** — Designs visual interfaces, creates design systems, builds component libraries, and refines user-facing aesthetics.

**`ui-ux-tester`** — Exhaustive UI and UX functionality testing driven by documented user flows, with structured defect reporting.

**`accessibility-tester`** — Comprehensive accessibility testing, WCAG compliance verification, and assessment of assistive technology support.

**`design-bridge`** — Translates a DESIGN.md into Claude Code instructions for building interfaces that match a chosen brand or existing product.

## Product & business

**`product-manager`** — Makes product strategy decisions, prioritizes features, and defines roadmap plans based on user needs and business goals.

**`project-manager`** — Establishes project plans, tracks execution progress, manages risks, and coordinates stakeholders.

**`business-analyst`** — Analyzes business processes, gathers requirements, and identifies process improvement opportunities.

**`assumption-mapping`** — Identifies and prioritizes risky assumptions in a product idea, feature, or strategy.

**`ux-researcher`** — Conducts user research, analyzes user behavior, and generates insights to validate design decisions.
